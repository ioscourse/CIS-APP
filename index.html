How do I test app?

1) Have latest version of iOS7 for your iPhone/iPad

Settings > General > Software Update

2) Send iPhone/iPad UDID

  • Download and install UDID Tool:

    https://itunes.apple.com/us/app/udid-tool/id377602501?mt=8

  • Run UDID Tool app and send UDID to: konkolapps@gmail.com

  • Wait for email from konkolapps@gmail.com before going to step 3.

3) Install & Test App (need latest version of iOS7)

  • Download and Install (Lastest Build: 10/7/2013)

  • Install via QR Scan
