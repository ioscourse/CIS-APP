How to test app:

1) If interested in testing above app, please install UDID Tool app to your iPhone/iPad

  • Download and install UDID Tool:

https://itunes.apple.com/us/app/udid-tool/id377602501?mt=8

  • Run UDID Tool app and send UDID to: konkolapps@gmail.com

  • Wait for email from konkolapps@gmail.com before going to step 2.

2) Install & Test App Download and install CIS App (iPhone/iPad)

  • Install via Link: https://ota.io/fogtzv

  • Install via Scan (Need SCAN APP to scan QR Code)

3) Post Feedback/Issues

  • https://mycisapp.uservoice.com/

    Items to test:

  • Does app open?

  • Can you navigate to pages?

  • Is screen layout easy to use?

  • What suggestions do you have on: Text, layout, colors, videos, content?
